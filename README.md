
# 🛒 ShopEasy - E-Commerce Website

ShopEasy is a full-stack E-Commerce web application developed using **Python Flask**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.

The project provides an online shopping experience where users can browse products, place orders, contact the store, and administrators can manage products and orders from an admin dashboard.

---

## 🚀 Features

### 👤 User Features
- Home Page
- Products Page
- Product Search
- Contact Page
- Checkout Page
- Order History
- Responsive UI
- Contact Form

### 🔐 Admin Features
- Admin Login
- Admin Dashboard
- Add Product
- Edit Product
- Delete Product
- Manage Orders
- View Contact Messages

---

## 🛠️ Technologies Used

- Python
- Flask
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Jinja2

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/shopEasy-project.git
```

### Go to Project Folder

```bash
cd shopEasy-project
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Import Database

Import **shopeasy.sql** into MySQL.

### Run Application

```bash
python app.py
```

Open Browser

```
http://127.0.0.1:5000/
```
